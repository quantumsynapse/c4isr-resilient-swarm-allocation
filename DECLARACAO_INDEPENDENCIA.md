# Declaração de Independência

Este projeto foi desenvolvido como experimento computacional independente.

Todos os agentes, ações, grafos, canais, perturbações, capacidades e resultados são sintéticos. Nenhum dado foi obtido de aeronave, organização militar, operador, sensor, enlace, missão ou sistema real.

A implementação não representa produto, doutrina, contrato ou requisito de qualquer instituição.
